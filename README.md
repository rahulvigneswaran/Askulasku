# Say in every language
Type any word or phrase and watch it morph through 30+ languages in a beautiful animation. Export as video to share anywhere.

## Features

- **30+ languages** — including major Indian languages: Tamil, Hindi, Bengali, Telugu, Malayalam, Marathi, Gujarati, Kannada, Odia, Punjabi
- **Two animation styles** — smooth morph transition or split-flap board effect
- **Adjustable pace** — from slow and cinematic to fast
- **Video export** — renders a clean MP4 with a watermark, ready to share as an Instagram Reel or WhatsApp status
- **Works offline** — all rendering happens in the browser, no server needed

## Usage

1. Type a word or phrase in the input box
2. Hit **Animate**
3. Watch your phrase appear in language after language
4. Click **Share** to export a video

## Tech

Pure HTML/CSS/JS — no frameworks, no build step. Video encoding via WebCodecs API with mp4-muxer. Translations via MyMemory API.
